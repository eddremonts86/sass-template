## üéØ Common Tasks

### Format Code

`Shift+Alt+F` (or auto on save)

### Run Linter

`Cmd+Shift+P` ‚Üí "Tasks: Run Task" ‚Üí "Lint: Check"

### Debug Application

Press `F5` and select configuration

### Run Tests

`Cmd+Shift+P` ‚Üí "Tasks: Run Task" ‚Üí "Test: Unit" or "Test: E2E"

### Use Code Snippet

Type prefix (e.g., `rfc`) and press `Tab`
